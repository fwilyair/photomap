Apple Look Around

这是 iOS 原生的街景体验，只能在 iPhone/iPad 上通过 MapKit 的 MKLookAroundViewController 调用。无法在网页中展示，但可以在 Apple 地图 App 中体验。

在 Apple 地图中打开 →

📱 原生集成

直接使用 MKLookAroundScene API，无需额外配置或 API Key。

🎨 体验一致

视觉风格与 iOS 系统完美融合，过渡动画流畅自然。

💰 免费

无 API 调用费用，无请求限制，无需注册开发者账号。

🌍 覆盖有限

目前主要覆盖欧美和日本部分城市，国内覆盖极少。
